Introduction to HTML

What is HTML?

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It uses a system of tags to define the structure and content of a webpage.

Key Concepts:

  • HTML documents are the foundation of web pages
  • Tags define different types of content
  • Attributes provide additional information

Basic HTML Document Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first web page.</p>
</body>
</html>

Each part serves a specific purpose:

  • !DOCTYPE html: Declares the document type
  • html: The root element
  • head: Contains metadata and links
  • body: Contains the visible content

HTML Elements & Attributes

<!-- Element with attributes -->
<a href="https://example.com" target="_blank">Visit Example</a>

<!-- Image with multiple attributes -->
<img src="photo.jpg" alt="A beautiful sunset" width="300">

Elements can have various attributes that:

  • Provide additional information
  • Control element behavior
  • Add accessibility features
HTML Elements

Text Elements

1. Headings

<h1>Main Title</h1>
<h2>Subtitle</h2>
<h3>Section Title</h3>
<h4>Sub-section Title</h4>
<h5>Minor Title</h5>
<h6>Smallest Title</h6>

Use headings to create a clear content hierarchy.

Text Formatting

<p>This is a paragraph with <strong>bold text</strong> and <em>italic text</em>.</p>

<p>This paragraph has a <br> line break.</p>

<p>This text has <mark>highlighted</mark> and <u>underlined</u> parts.</p>

Links and Images

<!-- External link -->
<a href="https://www.example.com" target="_blank">Visit Example.com</a>

<!-- Internal link -->
<a href="about.html">About Us</a>

<!-- Image with alt text -->
<img src="cat.jpg" alt="A cute cat" width="300" height="200">

<!-- Linked image -->
<a href="profile.html">
    <img src="avatar.jpg" alt="User profile">
</a>

Best Practices:

  • Always include descriptive alt text for images
  • Use meaningful text for links
  • Consider using target="_blank" for external links
