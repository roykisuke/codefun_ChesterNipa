Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a fundamental technology for web development. It allows you to control the presentation and layout of web pages. In this section, we will explore the basics of CSS, including how it works, how to apply styles, and how to use CSS selectors.
CSS Selectors

CSS Selectors

CSS selectors are used to select the HTML elements that you want to style. They target HTML elements based on their types, classes, IDs, attributes, etc.

Types of Selectors:

  • Universal Selector (*): Selects all elements.
  • Type Selector (p, h1): Selects all elements of the specified type.
  • Class Selector (.className): Selects elements with a specific class.
  • ID Selector (#idName): Selects an element with a specific ID.

Example:

/* Universal Selector */
* {
    margin: 0;
    padding: 0;
}

/* Type Selector */
h1 {
    color: #4CAF50;
}

/* Class Selector */
.text-center {
    text-align: center;
}

/* ID Selector */
#unique-element {
    font-size: 2em;
}
<h1 class="text-center">Hello World!</h1>
<p id="unique-element">This is a unique paragraph.</p>
CSS Box Model

CSS Box Model

The CSS box model describes the rectangular boxes generated for elements. It consists of the content, padding, border, and margin.

Box Model Parts:

  • Content: The actual content inside the element (e.g., text or images).
  • Padding: Clears space around the content inside the border.
  • Border: Surrounds the padding (optional).
  • Margin: Clears space outside the border.

Example:

.box {
    width: 200px;
    padding: 20px;
    border: 5px solid #4CAF50;
    margin: 30px;
}
<div class="box">This is a box model example!</div>
CSS Layout

CSS Layout

CSS layout is about positioning elements in your web page. There are several layout methods in CSS, including block, inline, flexbox, and grid.

Flexbox Layout Example:

Flexbox allows you to create complex layouts with ease.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    height: 100vh;
}

.box {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
    margin: 10px;
}
<div class="container">
    <div class="box"></div>
    <div class="box"></div>
    <div class="box"></div>
</div>
CSS Text Styling

CSS Text Styling

You can style text using CSS properties such as font-family, font-size, font-weight, line-height, text-align, etc.

Example:

.text-style {
    font-family: 'Arial', sans-serif;
    font-size: 20px;
    font-weight: bold;
    text-align: center;
    line-height: 1.5;
}
<p class="text-style">This is a styled text paragraph.</p>
CSS Fonts and Web Typography

CSS Fonts and Web Typography

Web typography allows you to use custom fonts from services like Google Fonts, or you can use system fonts.

Example (Using Google Fonts):

<head>
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>
body {
    font-family: 'Roboto', sans-serif;
}

Example (Fallback Fonts):

body {
    font-family: 'Arial', 'Helvetica', sans-serif;
}
CSS Backgrounds

CSS Backgrounds

You can set backgrounds for elements using the background property. Backgrounds can include colors, images, gradients, etc.

Example:

.container {
    background-color: #f4f4f4;
    background-image: url('background.jpg');
    background-repeat: no-repeat;
    background-size: cover;
    height: 400px;
    width: 100%;
}
<div class="container">
    <p>This is a container with a background image!</p>
</div>

Example (Gradient Background):

.container {
    background: linear-gradient(to right, #ff7e5f, #feb47b);
    height: 400px;
    width: 100%;
}
CSS Borders

CSS Borders

Borders are used to outline an element. You can set the border’s width, style, and color.

Example:

.box {
    width: 200px;
    height: 200px;
    border: 5px solid #4CAF50;
    padding: 20px;
}

Example (Rounded Borders):

.box {
    width: 200px;
    height: 200px;
    border: 5px solid #4CAF50;
    border-radius: 15px;
}

Example (Multiple Borders):

.box {
    width: 200px;
    height: 200px;
    border-top: 5px solid #4CAF50;
    border-right: 5px dashed #FF5733;
    border-bottom: 5px dotted #C70039;
    border-left: 5px double #900C3F;
}
