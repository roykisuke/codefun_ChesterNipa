Learn JavaScript's Fundamentals

JavaScript Syntax
Introduction to JavaScript

Objective:

Learn what JavaScript is and how to write basic JavaScript code.

Concept:

JavaScript is a programming language used to create dynamic and interactive effects on websites.

Sample Code:

// This is a simple JavaScript program
console.log("Hello, world!"); // Outputs to the console

Things to Learn:

  • Understanding the role of JavaScript in web development.
  • How to write a basic script.
  • How to use console.log() to print output to the console.
JavaScript Variables

Objective:

Understand how to declare and use variables.

Concept:

Variables in JavaScript are used to store data. They can be declared using var, let, or const.

Sample Code:

let name = "John"; // Declaring a variable 'name' with string value
const age = 30; // Declaring a constant 'age'
var isStudent = true; // Declaring a variable 'isStudent' with boolean value

console.log(name, age, isStudent);

Things to Learn:

  • Different ways to declare variables (let, const, and var).
  • The difference between let, const, and var.
  • How to store and access different types of data (string, number, boolean).
JavaScript Input and Output

Objective:

Learn how to get user input and display output.

Concept:

Use prompt() to get input from the user, and alert() or console.log() to display output.

Sample Code:

// Getting user input
let userName = prompt("What's your name?");
alert("Hello, " + userName + "!");

Things to Learn:

  • How to use prompt() to gather input from the user.
  • How to display data using alert() or console.log().
JavaScript Arithmetic Operators

Objective:

Understand and use arithmetic operators in JavaScript.

Concept:

Arithmetic operators are used for basic calculations such as addition, subtraction, multiplication, division, etc.

Sample Code:

let x = 10;
let y = 5;

let sum = x + y; // Addition
let difference = x - y; // Subtraction
let product = x * y; // Multiplication
let quotient = x / y; // Division
let remainder = x % y; // Modulo (remainder)

console.log("Sum:", sum);
console.log("Difference:", difference);
console.log("Product:", product);
console.log("Quotient:", quotient);
console.log("Remainder:", remainder);

Things to Learn:

  • How to use arithmetic operators (+, -, *, /, %).
  • The order of operations (PEMDAS/BODMAS).
JavaScript Comparison Operators

Objective:

Learn how to compare values using comparison operators.

Concept:

Comparison operators return a boolean value (true or false).

Sample Code:

let a = 5;
let b = 10;

console.log(a == b); // Equals
console.log(a != b); // Not equals
console.log(a > b); // Greater than
console.log(a < b); // Less than
console.log(a >= b); // Greater than or equal to
console.log(a <= b); // Less than or equal to

Things to Learn:

  • How to use comparison operators (==, !=, >, <, >=, <=).
  • The difference between == (loose equality) and === (strict equality).
JavaScript if-else Statements

Objective:

Learn how to make decisions in JavaScript using if, else if, and else statements.

Concept:

Conditional statements allow the program to execute different code based on certain conditions.

Sample Code:

let age = 18;

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Things to Learn:

  • How to use if, else if, and else for branching logic.
  • How to write conditions using comparison operators.
  • Understanding boolean expressions.
JavaScript for Loops

Objective:

Learn how to repeat actions using loops.

Concept:

A for loop allows code to be repeated a certain number of times.

Sample Code:

for (let i = 0; i < 5; i++) {
    console.log("Iteration number: " + i);
}

Things to Learn:

  • How to structure a for loop.
  • The role of the loop control variables (i in this case).
  • The purpose of the three parts of the loop (initialization, condition, increment).
JavaScript Arrays

Objective:

Learn how to work with arrays in JavaScript.

Concept:

Arrays are used to store multiple values in a single variable. They can hold any type of data.

Sample Code:

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Accessing the first element
fruits.push("Orange"); // Adding an element to the end of the array
console.log(fruits);

for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]); // Looping through the array
}

Things to Learn:

  • How to declare an array.
  • How to access array elements using indices.
  • How to use array methods like push(), pop(), shift(), and unshift().
  • Iterating over arrays with loops.
